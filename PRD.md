# PRD: at-tray v2

## Goal
Windows 系统托盘服务，自带调度引擎（不用 at/schtasks），支持灵活的定时任务管理。

## Core Features

### 托盘与后台
- 托盘常驻，服务跟随运行
- 托盘右键菜单：新建任务 / 任务列表 / 设置 / 退出
- 任务到达时执行操作 + 提前通知

### 新建任务窗口
- **时间选择**：快速预设 5/15/30min/1h/2h/4h + 自定义输入
- **动作**：关机 / 重启 / 锁定 / 命令（任意命令行）
- **重复**：不重复（默认）/ 每天 / 每周 / 自定义 Cron 式
- **最大次数**：=1 时隐藏重复选项
- **提前通知**：0/5/10/15/30 分钟前通知

### 任务列表
- 表格展示：时间 / 动作 / 重复 / 状态
- 删除 / 暂停 / 启用任务

## UX 原则
- 时间选择必须快：预设按钮优先
- 最少点击数完成任务创建
- 托盘气泡通知，不要弹窗打断

## Tech Stack
- Go（pure Win32 API + gogpu/systray）
- 纯 Go，零 CGo，零外部依赖
- JSON 存储于 `%APPDATA%/at-tray/`
- 单文件 exe

## Out of Scope
- schtasks/at 集成
- 网络/远程管理
- 多语言
- 主题
